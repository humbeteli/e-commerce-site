# E-Commerce Website

A modern e-commerce web application built with React, TypeScript, and DummyJSON API.

## Features

* Product listing
* Product details page
* Search products
* Add to cart
* Responsive design
* Category filtering

## Technologies Used

* React
* TypeScript
* CSS
* React Router
* DummyJSON API
